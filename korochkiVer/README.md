# Корочки.есть — тёмная версия интерфейса

Проект выполнен на PHP + MySQL.

## Запуск
1. Создайте базу данных MySQL вручную.
2. Импортируйте файл `database.sql` в созданную базу.
3. Укажите параметры подключения в `includes/config.php`.
4. Откройте `index.php` через локальный сервер.

## Данные администратора
- Логин: `Admin`
- Пароль: `KorokNET`

## Что реализовано
- регистрация пользователя;
- авторизация;
- создание заявки на обучение;
- личный кабинет со статусами заявок;
- отзыв только после завершения обучения;
- панель администратора с поиском, фильтрацией, пагинацией и изменением статуса;
- защита от HTML-тегов и скриптов в пользовательском вводе;
- метатеги `description`, `keywords`, Open Graph и Twitter Card;
- микроразметка Schema.org на главной странице;
- адаптивный интерфейс;
- слайдер с автопрокруткой 3 секунды.
